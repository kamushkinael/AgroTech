## Финансовое резюме проекта

* Капитальные затраты (CAPEX): 1 347 000 ₽ (Оборудование, датчики, ноутбук, монтаж).

* Экономия на ПО: около 400 000 – 700 000 ₽ за счет использования Open Source решений.
